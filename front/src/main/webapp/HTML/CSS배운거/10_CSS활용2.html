display 속성

  1. inline : 수평 방향 [ 자동 줄바꿈 X ]
    1. span, a, img 등
    2. 박스의 크기 적용이 안됨 [ width, height X ]
    3. 바깥여백 위/아래 적용 안됨
  2. block : 수평 행 전체 차지 [ 자동 줄바꿈 O ]
    1. div, p, ul, li 등
    2. 박스의 크기 적용이 됨
    3. 모든 바깥 여백 적용 O
  3. inline-block
    1. block 속성에 줄바꿈 X
