ul : unordered list : 순서 없는 목록 태그

ol : ordered list : 순서가 있는(순서번호) 목록 태그

li : list item : 목록 값

  • 어떤 생각을 일정한 형식에 따라 글자로 나타낸 것. 시·소설·수필·희곡 등을 비롯하여, 일기·편지·논설·신문 기사 등과 같이 일정한 내용을 담은 모든 기록을 두루 이르는 말임. 세는 단위는 줄·편."∼을 읽다"
  • 봄(영어: Spring)은 온대 지방의 사계절 중 하나이다. 봄철, 춘계(春季), 춘기(春期)라고도 한다.
  • 여름은 온대 지방의 사계절 중 하나이다. 하계(夏季), 하기(夏期), 하절(夏節), 여름철이라고도 하며, 여름날은 하일(夏日)이라고 한다.
  • 가을은 온대 지방의 사계절 중 하나이다. 가을철, 추계(秋季), 추기(秋期), 오추(梧秋)라고도 한다. 대한민국에서는 가을을 "갈"이라고 줄여 부르기도 했다.
